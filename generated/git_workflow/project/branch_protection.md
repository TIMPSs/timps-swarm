# Branch protection policy for `main`

- [ ] CI / lint

- [ ] CI / coverage ≥ 70%

## Pull-request reviews
- [ ] Require **1** approving review(s).

- [ ] Enforce for admins: **True**.

- [ ] Require signed commits: **False**.
- [ ] Block force pushes.
- [ ] Block branch deletion.
